戦闘シーンクラス

シーン基本クラス(CScene)を派生したクラス。
Init、Control、Draw、Render関数がライブラリのサブルーチン側で呼ばれるので、宣言すること。
今回は2D描画のみの予定なのでRender関数は宣言をしているが、中身はない。
